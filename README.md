# 💻 TechFix - Serwis Komputerowy

![TechFix Logo](images/logo.png)

Nowoczesna strona internetowa dla fikcyjnego serwisu komputerowego **TechFix**.  
Projekt został wykonany jako portfolio frontendowe, prezentujące możliwości tworzenia responsywnych stron internetowych z wykorzystaniem HTML, CSS oraz JavaScript.

---

## 🚀 Demo

Strona może zostać uruchomiona przez GitHub Pages:
https://YoungpiotreK.github.io/Sklep-TechFix/


---

# 📌 Funkcje

✅ Nowoczesny design typu dark mode  
✅ Responsywność na komputery, tablety i telefony  
✅ Animacje podczas przewijania strony  
✅ Animowane liczniki statystyk  
✅ Mobilne menu nawigacyjne  
✅ Sekcja usług  
✅ Sekcja realizacji  
✅ Cennik usług  
✅ FAQ  
✅ Formularz kontaktowy  
✅ Efekty świetlne i gradienty technologiczne
